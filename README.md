# My Portfolio Website

A modern space-themed portfolio website built with Next.js 14, featuring interactive animations and a sleek design.

## Technologies Used

- Next.js 14
- React
- Framer Motion
- Tailwind CSS
- Three.js

## Features

- Interactive 3D animations
- Responsive design
- Modern UI/UX
- Project showcase
- Skills section
- Education timeline
- Contact information

## Project Structure

```bash
portfolio/
  |- app/               # Next.js app directory
  |- components/        # React components
  |- config/           # Configuration files
  |- constants/        # Constant values
  |- lib/             # Utility functions
  |- public/          # Static assets
```

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This website is optimized for deployment on Netlify. The build process is configured in `next.config.js` for static export.
